Type Definition pitches::midi::MidiPC[−][src]

pub type MidiPC = Pitch<MidiIC>;
Expand description

MIDI pitch classes.

Represents chromatic pitch classes as integers from 0 (C) to 11 (B).

Since pitch classes are circular, they have no musically interpretable order. The Ord instance only provides an arbitrary ordering from C to B for convenience.
